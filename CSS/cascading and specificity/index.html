Cascading and Specificity

  1. Position and Order of Appearence: The browser applies the styles appearing at the end.
  2. Origin:
    • User agent styles: The styles applied by browser to your html element by default.
    • Local user styles: The styles which are applied by OS.
    • Authored Styles: The CSS files that you have written.
    • Authored CSS! important
    • Local User Styles !important
    • User Agent Styles !important
  3. Specificity: The algo which tells which selector has more priority based on specificity score
  4. Importand: Highest Priority
