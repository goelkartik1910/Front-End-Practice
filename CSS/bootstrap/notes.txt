Bootstrap applies some css

container --> makes it container
container-fluid --> does the same thing but no margin
border --> applies border
border-2 --> makes it 2px border
row --> 12 columns in a row
col --> makes col in row also by default it will take all the width
col-3 --> it will take 3 parts of 12 columns
justify --> helps to add flex properties
col-lg-3 --> takes 3 columns only when the screen is larger than lg
col-lg-3 col-md-4 col-6 --> lg or more take 3, md or more it will be 4, other 6
my-5 --> gives a margin
btn-group --> it makes a group 

form-control --> input of form
form-label --> label for form
offset --> helps us to give spaces in row
form-text --> under the input text

card --> makes a card
card-header --> header of the card
card-body --> body of the card
card-footer --> footer of the card
